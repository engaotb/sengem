# 📱 Senjem Mobile - لعبة الأسئلة الهندسية للموبايل

نسخة متوافقة مع الموبايل من لعبة الأسئلة الهندسية Senjem، مصممة خصيصاً للشاشات الصغيرة والأجهزة المحمولة.

## 🌟 المميزات

### 📱 **تصميم متوافق مع الموبايل:**
- واجهة مخصصة للشاشات الصغيرة
- أزرار كبيرة وسهلة اللمس
- تنقل سلس بين الشاشات
- قائمة جانبية للفرق والإحصائيات

### 🎮 **تجربة مستخدم محسنة:**
- شاشة اختيار الفرق
- شاشة التخصصات مع صور
- شاشة الأسئلة مع صور كبيرة
- أزرار صحيح/خاطئ واضحة

### 📊 **إدارة الفرق:**
- 5 فرق مع نظام نقاط
- حفظ تلقائي لحالة اللعبة
- إحصائيات مفصلة
- إعادة تعيين اللعبة

## 🚀 كيفية الاستخدام

### **1. اختيار الفريق:**
- اضغط على الفريق الذي تريد اللعب به
- سيظهر الفريق النشط في الأسفل

### **2. اختيار التخصص:**
- اختر من بين 6 تخصصات هندسية
- كل تخصص يحتوي على 3 أسئلة (200، 400، 600 نقطة)

### **3. الإجابة على الأسئلة:**
- اضغط على "إظهار الإجابة" لرؤية الإجابة
- اضغط "صحيح" أو "خاطئ" لتسجيل النتيجة

### **4. القائمة الجانبية:**
- اضغط على أيقونة القائمة لرؤية الفرق والإحصائيات
- اضغط على أيقونة إعادة التعيين لبدء لعبة جديدة

## 🛠️ التقنيات المستخدمة

- **HTML5**: هيكل الصفحة المتوافق مع الموبايل
- **CSS3**: تصميم متجاوب مع Media Queries
- **JavaScript**: منطق اللعبة المحسن للموبايل
- **LocalStorage**: حفظ حالة اللعبة
- **Lucide Icons**: أيقونات جميلة

## 🌐 التشغيل

### الطريقة الأولى: مباشرة
افتح ملف `index.html` في متصفح الموبايل

### الطريقة الثانية: خادم محلي
```bash
python3 -m http.server 8000
```
ثم اذهب إلى `http://localhost:8000`

### الطريقة الثالثة: GitHub Pages
المشروع جاهز للتشغيل على GitHub Pages
